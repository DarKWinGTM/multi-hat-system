# Worked Example Shapes

These are response-shape guides, not rigid templates.

---

## Advisory Example Shape

- task summary
- developer view
- security view
- architect view
- synthesis
- next steps

---

## Decision Example Shape

- decision context
- options
- developer comparison
- security comparison
- architect comparison
- recommendation
- when to choose differently

---

## Clarify-First Example Shape

- task summary
- missing high-impact context
- 1-3 focused clarifying questions
- note that recommendation strength depends on the missing answers

---

## Migration / Rollout Advisory Shape

- current state
- target state
- developer rollout concerns
- security rollout concerns
- architecture sequencing concerns
- recommended phase path
- verification and rollback focus

---

## Technology Choice Comparison Shape

- decision context
- option set
- developer trade-offs
- security trade-offs
- architecture trade-offs
- decisive tension
- winner and flip conditions
